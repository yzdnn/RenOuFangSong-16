**～项目概述～**

「人偶仿宋 16」是基于「KHドットフォントシリーズ」中的「KHドット人形町16」增补简繁体汉字的 16px 点阵字体，适合在像素风格的游戏种作为装饰性手写体使用。

字体名的来源是日本真实地名“人形町”中的“人形”，中文一般叫人偶。

虽然原字体的风格描述是「教科书体」，但群友的反应都是“更像仿宋”。而且原字体消灭了所有斜捺，取而代之的是点画，没有教科书体会真的这么做，所以此项目决定使用「仿宋」这个名字。

再加上px的数16，于是就有了「人偶仿宋 16」这个名称。

目前本字体项目仍未完工，欢迎各位网友提出宝贵意见。
